# Firestorm - Tower defense

Tower defense - Creeps will be running along side a path, the player will build towers to defend the creeps from
reaching the end of the path.

Examples of existing TD: BloonTD 6

## Tech

This is the stack that will be used to deliver this game.

- React
- Phaser
- Supabase
- Vercel

## Getting started

```bash
npm install

npm run dev                  # http://localhost:3000
```
